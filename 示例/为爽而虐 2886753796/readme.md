# 为爽而虐 文档 2.4.34

## 内置改动

诅咒道具拿到手上后都会立即返回物品栏,诅咒饰品死亡不掉落,诅咒道具在换人时不会掉落,换人后继续保留

### 容器整理收集按钮

点整理按钮,排序该容器内道具,背包双击会混合排序物品栏和背包内道具
点跨整按钮,背包会混合排序物品栏和背包内道具,携带容器混合排序所在容器内的所有同类容器内道具,地面容器会混合排序周围同名容器内道具,不会跨船
点锁定/解锁按钮,锁定/解锁该容器只能放置当前已有的道具,重载游戏时已锁定容器继续会保持锁定,但显示按钮文本为锁定,不过不影响使用
点收集按钮,携带容器收集携带的容器内同名道具,地面容器收集周围容器内的同名道具,不会跨船,佩戴或手持懒人护符点按钮,则拾取周围地上容器内同名道具
换装按钮,仅妥协模组衣柜,原版衣柜功能
穿越按钮,仅各种末影箱,与随机其他世界的该容器交换道具

### 道具快捷收集存放

制作栏右键道具,或左键右键材料,从附近容器收集需要的道具或材料
物品栏按住CTRL右键道具,将该道具存放到附近非分解类的容器,不能存放溯源表裂缝表

## 暗影世界(主题1)

可选生物有1~3个暗影分身,为0则本主题关闭
玩家/大部分暗影生物/月影生物/不能移动的生物没有暗影分身
暗影分身免疫草织陷阱捕鸟陷阱捕虫网,受到致命伤害后消失不会有掉落物,主要是为了增加击败本体的作战难度而生
刷新模式,BOSS或非BOSS生物是否有暗影分身,以及暗影分身在本体特定血量时才出现,切斯特哈奇伯尼阿比盖尔必定有分身,切斯特哈奇分身是末影箱
再生模式,暗影分身受到致命伤害消失后的再生时间
护主模式,暗影分身是否吸引本体敌人的仇恨,没有攻击力的生物分身默认吸引
索敌模式,暗影分身是否始终跟随本体仇恨,默认仅分身没有仇恨时共享本体仇恨
暴动玩法,默认关闭,暗影分身都是暴动暗影,都会仇恨玩家
回归机制,玩家远离本体后回归,玩家远离分身后分身回归,默认BOSS的分身仅在玩家远离BOSS本体200距离后回归(每4距离为1地皮)

## 简单模式(主题2)

### 建筑改动

科学机器,给材料会精炼,给成品会消耗材料制作成品,给齿轮升级9次,升级提高给成品制作成品的速率
炼金引擎,有威尔逊的炼金术,给材料会消耗材料进行威尔逊的炼金,给月岩会消耗大理石炼成月岩,不能炼成彩虹宝石紫宝石,也没有月亮暗影转化术,可选像猪王一样兑换金块
灵子分解器,可以分解给它的道具并返还道具的至多50%的材料,耐久越高返回越高,没有保底返还,给绿宝石升级2次后可以返回至多100%的材料,连续给堆叠道具会全部分解堆叠道具
暗影操控器,给4噩梦燃料会至少20天内解锁麦斯威尔的专属制作配方和暗影秘典高礼帽使用能力,但一般同时只能召唤1个影子,老麦则会读书理智消耗降低到1
土地夯实器,可以像蚁狮一样兑换沙之石但兑换速度更快
钓具容器,可选将其变成一个20格0.15倍腐烂速率的末影箱,可以容纳钓具箱可以容纳的道具和制作好的菜肴,可选像鱼人王一样兑换海鱼
制图桌,可以给与其一个道具消耗一个草纸,获得该道具的学习蓝图

### 机制改动

危险时也能入睡,入睡1秒后摆脱敌人仇恨
用燃烧的火焰烹饪,比如猪人火炬,冰篝火,燃烧的树木,甚至手持的火把
净化怪兽肉为肉,用蜘蛛腺体蚊子血袋药膏等治疗道具净化怪兽肉,烤怪兽肉,怪兽肉干,怪兽小肉,净化概率为治疗血量/肉的掉血量
作祟地上的告密的心复活,该复活不会有血量上限惩罚
建造护符修理道具,用建造护符直接修补道具的耐久
额外可以直接材料修理草甲木甲影甲大理石甲鳞甲影刀,携带建造护符时可以直接材料修理玻璃刀玻璃斧铥矿三级套斧镐,斧镐和铥矿棒材料包括金子
格罗姆会吃周围昆虫,蜜蜂蝴蝶蚜虫月熠等

### 道具改动

甲板照明灯可以装备在头部,并常规添加燃料,耐久越高照明越高
钓具箱可以容纳角色专属道具,永久保鲜,可以存放孢子和蜘蛛
启迪之冠根据放入的每个孢子提高属性,红绿蓝孢子依次是60保暖,20%防水,60隔热,放5个红孢子后变热源携带生效,放5个绿孢子后绝缘携带即生效,放5个蓝孢子后变冷源携带生效
红绿蓝蘑菇帽现在依次有240保暖,100%防水,240隔热,不新鲜后该属性变弱
眼球伞现在在下雨时恢复耐久,下雪时3倍恢复耐久,放地上也回,酸雨无效,
